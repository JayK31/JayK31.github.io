Meet Me

What I Am
I'm a full stack web developer proficient in Ruby on Rails and JavaScript. I've built numerous web applications using technologies such as jQuery, D3, Backbone, PostgresSQL/MySQL, HTML5/CSS3, Bootstrap, and Skeleton. I have experience with TDD/BDD, and testing suites such as RSpec, Capybara, and Jasmine. I use git for version control. I'm a fan of Agile development, and I love working with APIs contributing to open source projects.
